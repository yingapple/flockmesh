# Roadmap

## Phase 0: Foundation (Now)

- Define architecture and domain model
- Scaffold repository and contribution guidelines
- Build first local runnable control-plane prototype

## Phase 1: Core Runtime

- Agent registry CRUD
- MCP connector abstraction
- Workflow engine with approval node
- Structured audit pipeline

## Phase 2: Team Adoption Loop

- Multi-project workspaces
- Role templates (sales, support, ops)
- Execution dashboard + replay UI

## Phase 3: Federation

- A2A-compatible delegation across org boundaries
- Cross-tenant policy trust contracts
- Plugin marketplace for community connectors

## Success Criteria

- A team can deploy within one day
- A workflow can be traced end-to-end in under one minute
- Sensitive actions are blocked by policy without custom coding
